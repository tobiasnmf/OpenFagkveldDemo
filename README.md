# OpenFagkveldDemo

Demo-repo for OpenClaw / fagkveld.

## Hva er dette?

Dette prosjektet er satt opp som et enkelt demo-repo for å vise hvordan OpenClaw kan brukes i praksis.

Typiske ting som kan demonstreres her:

- opprettelse og redigering av filer
- arbeid med Git og GitHub
- automatisering via CLI og agenter
- samarbeid gjennom commits og pull requests

## Kom i gang

Klone repoet:

```bash
git clone https://github.com/tobiasnmf/OpenFagkveldDemo.git
cd OpenFagkveldDemo
```

## Videre arbeid

Mulige neste steg:

- legge inn eksempelapplikasjon
- legge til demo-skript eller oppgaver
- dokumentere fagkveldens agenda
- vise agentdrevet utviklingsflyt
